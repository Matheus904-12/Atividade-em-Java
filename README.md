# ☕ Java Foundations — Estudos e Consolidação

> Branch dedicada ao aprendizado contínuo de Java.  
> Aqui cada arquivo, cada classe, cada exercício tem um propósito: **fixar os fundamentos e evoluir progressivamente**.

---

## 🎯 Objetivo

Consolidar os conceitos da linguagem Java por meio de prática constante, organizando os estudos em módulos temáticos. O foco não é apenas fazer funcionar — é **entender o porquê** de cada estrutura, padrão e decisão.

---

## 📚 Conteúdos Planejados

### 1. Fundamentos da Linguagem
- Tipos primitivos e Wrappers (`int`, `Integer`, `double`, `Double`, etc.)
- Operadores aritméticos, relacionais e lógicos
- Estruturas de controle: `if/else`, `switch`
- Laços de repetição: `for`, `while`, `do-while`
- Arrays e matrizes

### 2. Orientação a Objetos (POO)
- Classes e Objetos
- Construtores e sobrecarga
- Encapsulamento: `private`, `public`, `protected`, getters e setters
- Herança e `super`
- Polimorfismo (sobrecarga e sobrescrita)
- Classes abstratas e Interfaces
- Modificador `final` e `static`

### 3. Coleções e Estruturas de Dados
- `ArrayList`, `LinkedList`
- `HashMap`, `HashSet`
- Iteração com `for-each` e `Iterator`
- Comparação com `Comparable` e `Comparator`

### 4. Tratamento de Exceções
- `try`, `catch`, `finally`
- Exceções verificadas vs não verificadas
- Criando exceções customizadas

### 5. Entrada e Saída de Dados
- Leitura com `Scanner`
- Formatação de saída com `printf` e `String.format`
- Introdução a leitura/escrita de arquivos (`File`, `BufferedReader`, `BufferedWriter`)

### 6. Generics e Boas Práticas
- Métodos e classes genéricas
- Convenções de nomenclatura Java
- Princípios SOLID (introdução)
- Javadoc e comentários úteis

### 7. Extras e Desafios
- Algoritmos de ordenação (Bubble Sort, Selection Sort)
- Recursividade
- Mini projetos console (sistema de estoque, cadastro simples, calculadora OO)

---

## 📌 Regras desta Branch

- Cada tópico fica em sua própria pasta
- Todo código deve estar comentado explicando **o que faz e por quê**
- Commits frequentes com mensagens descritivas
- Sem pressa — o objetivo é **entender**, não apenas copiar
